# Human Escalation

In this module, you'll configure your AI agent to recognize when it needs to escalate complex customer requests to human agents.

## Overview

You will learn how to:
- Configure the Escalate tool for human handoff
- Set up Step-by-step Guides for human agents
- Build escalation logic into your contact flow
- Ensure smooth transitions from AI to human agents

## Prerequisites

Complete Module 3: Handling Customer Questions before starting this module.
